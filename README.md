# QR code reader

It's a simple QR code reader by using [jsQR](https://github.com/cozmo/jsQR).



## Features

- 



## Requirement

- Node.js >= 10.x
- npm >= 6.x



## Usage

```bash
# install node_modules
yarn install

# run locally
yarn start

# development build
yarn run build:d

# production build
yarn run build:p

```
